Links

Teste de Link(Este espaço pode ser um texto ou uma imagem)

a = tag âncora = link

atributo "href" = o link da página para a qual você deseja "encaminhar" o usuário

atributo title = título do site

atributo target = qual a janela de destino ao clicar no link. Ex: _self recarrega na própria página que a pessoa está

Imagens

tag img = imagem

Tem o fechamento inline (não tem outra tag de fechamento)

atributo "src" = o link da imagem que você quer buscar na pasta ou na web e colocar no seu site

atributo "alt" = Nome alternativo - para mecanismos de busca encontrar seu site e leitores de texto para necessidades especiais

atributo "title" = descrição que aparece quando você para o mouse em cima da imagem
