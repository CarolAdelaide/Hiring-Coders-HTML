Listas

ul = unordered list

li = list item

ol = ordered list

Listas desordenadas

Exemplo:

  • Item 1
  • Item 2

Atributo "type" = permite escolher se prefere círculo preenchido ou vazio para tópico de listagem

Listas ordenadas

Exemplo:

  1. Item 1
  2. Item 2

Atributo "start" = permite indicar de qual número você quer iniciar a contagem da lista

Atributo "type" = permite escolher se prefere número, letra maiuscula ou minuscula para a contagem

Listas de definição

Exemplo:

Palavra 1
Explicação da palavra 1

Lista de definição pode ser usada para dicionário, por exemplo

TABELAS

Tag table indica que faremos uma tabela

Tag thead = cabeçalho da tabela

Tag Tbody = o corpo da tabela

Tfoot = rodapé da tabela

Tr = table row = linha

Th = cabeçalho da tabela

Td = itens de preenchimento da tabela

Atributo "Colspan" é usado para juntar duas "células" da tabela - horizontalmente (mesclar no excel, por exemplo)

Atributo "Rowspan" é usado para juntar duas "linhas" da tabela - verticalmente (mesclar no excel, por exemplo)

Exemplo:

Mês da Atividade Nome da Atividade
Janeiro Leitura
Fevereiro Meditação
Total de atividades 2
